# 如何使用补丁

    - 前往 https://www.marcrobledo.com/RomPatcher.js/

    - ROM file 选项：请选择原韩版或者原美版英文版金银 GBC ROM。**不论选择使用哪个语言的 ROM，最终打完补丁的版本均为韩版汉化。**
        关于原版 ROM 的信息：
        韩版：
        金：Pocket Monsters Geum (Korea).gbc	
        MD5 82bd1d9171e60f147d9eeea13ef07a12
        银：Pocket Monsters Eun (Korea).gbc
        MD5 f5e14b069eaddee42735572bd23f7ef8

        美版：
        金：Pokemon - Gold Version (UE) [C][!].gbc
        MD5 a6924ce1f9ad2228e1c6580779b23878
        银：Pokemon - Silver Version (UE) [C][!].gbc
        MD5 2ac166169354e84d0e2d7cf4cb40b312

    - Patch file 选项：请选择对应用于原韩版或者原美版英文版的金银 ROM 的 ips 补丁。例：如果原 ROM 是英文版宝可梦金，就选择压缩包内的 ForUSA/USA_Gold_CHS.ips 。

    - 点击 Apply Patch 即可下载到打完补丁的汉化 ROM。
